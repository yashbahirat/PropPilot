<task id="4">
<action>
Initialize Prisma in the project:

```bash
npx prisma init --datasource-provider postgresql
```

This creates `prisma/schema.prisma` with a basic PostgreSQL datasource and `prisma/.env` (which we'll ignore — env vars go in `.env.local`).

Update `prisma/schema.prisma` datasource to use both DATABASE_URL and DIRECT_URL:

```prisma
datasource db {
  provider  = "postgresql"
  url       = env("DATABASE_URL")
  directUrl = env("DIRECT_URL")
}

generator client {
  provider = "prisma-client-js"
}
```
</action>
<read_first>
- `package.json` (verify prisma is installed before running init)
</read_first>
<acceptance_criteria>

- `prisma/schema.prisma` contains `directUrl = env("DIRECT_URL")`
- `prisma/schema.prisma` contains `provider = "prisma-client-js"` in generator block
</acceptance_criteria>
</task>
