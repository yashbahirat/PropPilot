# 02-03: Firm Create/Edit Form

## Goal
Implement a tabbed form to create and update Firm records.

## Context
- User chose a Tabbed Interface for the Firm edit form.
- The Firm model has many fields. We will divide them into "General" (name, slug, logoUrl, description, URLs), "Metrics" (account sizes, fees, profit targets, drawdowns), and "Settings" (isActive, isFeatured, isVerified, lastVerifiedAt).
- Logo image handling: User chose simple URL input.
