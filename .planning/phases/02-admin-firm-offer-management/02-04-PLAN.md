# 02-04: Offer Management (Inline Tab)

## Goal
Manage discount codes and offers for a specific firm.

## Context
- The user chose to manage offers inline as a tab inside the Firm edit page.
- Requires fetching `FirmOffer` records associated with the current `firmId`.
